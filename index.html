Comment ça marche ?

  1. GitHub Pages lit le fichier index.html.
  2. Ce fichier contient le HTML de la page.
  3. Le CSS dans la balise <style> ajoute la mise en forme.
  4. Quand vous modifiez le fichier et poussez le commit, la page se met à jour.
